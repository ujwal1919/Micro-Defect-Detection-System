# HARDWARE & SOFTWARE REQUIREMENT

## SOFTWARE REQUIRMENTS

### **Python**
Primary language for deep learning model development, backend API logic, and image processing.

### **PyTorch**
Deep learning framework for creating, training, and deploying CNN models (ResNet18, ResNet50) for defect detection.

### **TorchVision**
Pre-trained models library providing ResNet architectures with ImageNet weights for transfer learning.

### **OpenCV**
For preprocessing, analyzing PCB images, edge detection, contour analysis, and computer vision algorithms.

### **FastAPI**
Modern Python web framework for building RESTful API endpoints and handling image upload requests.

### **Uvicorn**
ASGI web server for running the FastAPI backend application with high performance.

### **NumPy & Pandas**
Essential for handling image data arrays, numerical computations, and statistical analysis.

### **Pillow (PIL)**
Python Imaging Library for image manipulation, format conversion, and preprocessing operations.

### **Next.js**
React-based framework for building the frontend web application with server-side rendering capabilities.

### **React**
JavaScript library for building interactive user interfaces and component-based frontend development.

### **TypeScript**
Type-safe programming language for frontend development ensuring code reliability and maintainability.

### **Tailwind CSS**
Utility-first CSS framework for rapid UI development and responsive design.

### **OpenCV-Python**
Python bindings for OpenCV providing computer vision algorithms for defect detection analysis.

### **Chart.js & Recharts**
JavaScript charting libraries for result visualization and defect analytics dashboard.

### **Python-multipart**
Library for handling file uploads in FastAPI, enabling image file processing from frontend.
