SOFTWARE REQUIREMENTS FOR PCB DEFECT DETECTION PROJECT
========================================================

1. Python 3.13+
   - Primary programming language for backend and AI/ML development

2. PyTorch 1.9.0+
   - Deep learning framework for CNN model training and inference

3. TorchVision 0.10.0+
   - Pre-trained models library (ResNet18, ResNet50)

4. OpenCV 4.5.3+
   - Computer vision library for image preprocessing and analysis

5. FastAPI 0.68.1+
   - Web framework for RESTful API development

6. Uvicorn 0.15.0+
   - ASGI web server for running FastAPI

7. NumPy 1.21.2+
   - Numerical computing library

8. Pillow 8.3.2+
   - Image processing library

9. Node.js 18.0+
   - JavaScript runtime for frontend

10. Next.js 13.5.1+
    - React framework for frontend development

11. React 18.2.0+
    - JavaScript library for UI development

12. TypeScript 5.2.2+
    - Type-safe programming language

13. Tailwind CSS 3.3.3+
    - CSS framework for styling

14. Chart.js 4.4.7+
    - Charting library for data visualization
